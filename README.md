# Samsung Wireless Audio Multiroom app

Use this app to control your Samsung soundbar.

## Device: Samsung Wireless Audio Multiroom

#### Triggers

- Turned on
- Turned off
- Volume changed

#### Conditions

- Is on

#### Actions

- Turn on
- Turn off
- Toggle
- Set volume
- Increase volume
- Decrease volume
- Mute 
- Unmute
- Set input source
- Play audio url
